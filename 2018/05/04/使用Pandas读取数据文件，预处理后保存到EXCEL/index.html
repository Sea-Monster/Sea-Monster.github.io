使用Pandas读取数据文件，预处理后保存到EXCEL

发表于 2018-05-04

1. 读取数据文件，转化为DataFrame对象

读取文件

一般来说是csv后缀的文件，当然也可能是其他后缀名，里边的内容可以直接当作文本文件打开，例如

或者

我们用Python的Pandas库处理读取这些数据，首先读取数据文件
import pandas as pd
data_frame = pd.read_table(文件路径, header=None, error_bad_lines=False, sep=’|’)
其中，

  • header如果不设置为None，默认第一行会作为header（也就是列名）
  • error_bad_lines设置为False，是为了有些不合规则的行不会导致整个导入工作终止
  • sep就是设置分隔符，默认是\t

指定列名

因为另外有一份专门的文件记录了这些数据对应的字段，

现在我们把这些字段名设置为DataFrame对象的列名。
data_frame.columns = columns

2. 保存到EXCEL，以我们惯常看到的格式。

data_frame.to_excel(需要输出到的文件的全路径)
